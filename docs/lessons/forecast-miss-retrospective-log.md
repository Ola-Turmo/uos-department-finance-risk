## Categories of Forecast Misses

### 1. Model Errors
- Incorrect model selection
- Poor parameter tuning
- Inadequate feature set

### 2. Data Quality Issues
- Missing historical data
- Incorrect historical data
- Data processing errors

### 3. Assumption Failures
- Wrong macroeconomic assumptions
- Incorrect business trajectory assumptions
- Missing competitive intelligence

### 4. One-Time Events
- Non-recurring items not anticipated
- External shocks (pandemics,自然灾害)
- Policy changes

### 5. Execution Variance
- Actual business decisions diverged from plan
- Execution not matching strategy
- Resource constraints

## Review Frequency
- Weekly: Quick assessment of current week misses
- Monthly: Deep dive into monthly forecast accuracy
- Quarterly: Full retrospective with methodology review
- Annually: Comprehensive forecast model evaluation

## Key Metrics to Track
- Mean Absolute Percentage Error (MAPE)
- Mean Absolute Scaled Error (MASE)
- Directional Accuracy
- Bias Assessment
